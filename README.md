# JewelPack

Next.js UI for a jewellery stock + sales management app.

## Run

```powershell
npm install
npm run dev
```

Open `http://localhost:3000`.

## Pages

- Dashboard: `/dashboard`
- Stock: `/stock` (Add: `/stock/new`)
- Sales: `/sales` (New: `/sales/new`)
- Purchases: `/purchases`
- Customers: `/customers`
- Suppliers: `/suppliers`
- Reports: `/reports`
- Settings: `/settings`
